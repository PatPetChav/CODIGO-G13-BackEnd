pasos para configurar basica de express con mysql
1) npm init -y
2) npm install express mysql dotenv (para manejo de variables de entorno)
3) npm install --save-dev nodemon
4) .gitignore  node_modules/
5) crear script en package.json
 "scripts": {
    "dev": "nodemon index.js",
    "start": "node index.js",
    "test": "echo \"Error: no test specified\" && exit 1"
  },
6) npm run dev (para ejecutar)
